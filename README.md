Developer Portfolio

Quick start

- Open `index.html` in your browser, or serve the folder with a simple HTTP server.

Local server (Python 3):

```bash
python -m http.server 8000
```

Customization

- Replace text and the avatar image in `index.html`.
- Update styles in `css/styles.css` and scripts in `js/script.js`.

Deploy

- Host on GitHub Pages, Netlify, Vercel, or any static host.
